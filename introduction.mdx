## What's Inside

Vicenta comes with a carefully crafted foundation:

- **Modern React 18**: Leverage the latest React features including Server Components
- **App Router**: Take advantage of Next.js 14's improved routing system
- **Tailwind CSS**: Rapidly build custom user interfaces with utility-first CSS
- **TypeScript**: Catch errors early with static type checking
- **Font Optimization**: Automatic optimization and loading of Google Fonts
